<h1>Synopsis:</h1>

<p>The goal of this report is to use raw data from sensors worn by the user and attached to the equipment, to predict the exercise class [A,B,C,D,E]. The sensors are:</p>

<pre><code>1. roll, pitch &amp; yaw of users belt

2. X, Y &amp; Z axis from gyro on users belt

3. X, Y &amp; Z axis from accelerometer on users belt

4. X, Y &amp; Z axis from magnets on users belt

5. roll, pitch &amp; yaw of users arm

6. X, Y &amp; Z axis from gyro on users arm

7. X, Y &amp; Z axis from accelerometer on users arm

8. X, Y &amp; Z axis from magnets on users arm

9. roll, pitch &amp; yaw of dumbbell

10. X, Y &amp; Z axis from gyro on dumbbell

11. X, Y &amp; Z axis from accelerometer on dumbbell

12. X, Y &amp; Z axis from magnets on dumbbell

13. roll, pitch &amp; yaw of users forearm

14. X, Y &amp; Z axis from gyro on users forearm

15. X, Y &amp; Z axis from accelerometer on users forearm

16. X, Y &amp; Z axis from magnets on users forearm</code></pre>

<div id="loading-the-source-data" class="section level2">

<h2>Loading the source data:</h2>

<pre class="r"><code>library(dplyr)

library(caret)

 

train_df &lt;- read.csv( &quot;https://d396qusza40orc.cloudfront.net/predmachlearn/pml-training.csv&quot; )

test_df &lt;- read.csv( &quot;https://d396qusza40orc.cloudfront.net/predmachlearn/pml-testing.csv&quot; )</code></pre>

</div>

<div id="preprocessing-the-data" class="section level2">

<h2>Preprocessing the data:</h2>

<p>Note, the train_df data has the outcome variable classe, while the test_df does not. For the purpose of model building and testing, a subset of data containing the outcome, must be created. The following code subdivides the original training data into two subsets:</p>

<pre class="r"><code>set.seed(3433)

inTrain = createDataPartition( train_df$classe, p = 3/4)[[1]]

training = train_df[ inTrain,]

testing = train_df[-inTrain,]</code></pre>

<p>This section of code creates a formula with classe as the outcome and all raw sensor data as predictors:</p>

<pre class="r"><code>raw_sensor_names &lt;- c(&quot;roll_belt&quot;, &quot;pitch_belt&quot;, &quot;yaw_belt&quot;,

&quot;gyros_belt_x&quot;, &quot;gyros_belt_y&quot;, &quot;gyros_belt_z&quot;,

&quot;accel_belt_x&quot;, &quot;accel_belt_y&quot;, &quot;accel_belt_z&quot;,

&quot;magnet_belt_x&quot;, &quot;magnet_belt_y&quot;, &quot;magnet_belt_z&quot;,

&quot;roll_arm&quot;, &quot;pitch_arm&quot;, &quot;yaw_arm&quot;,

&quot;gyros_arm_x&quot;, &quot;gyros_arm_y&quot;, &quot;gyros_arm_z&quot;,

&quot;accel_arm_x&quot;, &quot;accel_arm_y&quot;, &quot;accel_arm_z&quot;,

&quot;magnet_arm_x&quot;, &quot;magnet_arm_y&quot;, &quot;magnet_arm_z&quot;,

&quot;roll_dumbbell&quot;, &quot;pitch_dumbbell&quot;, &quot;yaw_dumbbell&quot;,

&quot;gyros_dumbbell_x&quot;, &quot;gyros_dumbbell_y&quot;, &quot;gyros_dumbbell_z&quot;,

&quot;accel_dumbbell_x&quot;, &quot;accel_dumbbell_y&quot;, &quot;accel_dumbbell_z&quot;,

&quot;magnet_dumbbell_x&quot;, &quot;magnet_dumbbell_y&quot;, &quot;magnet_dumbbell_z&quot;,

&quot;roll_forearm&quot;, &quot;pitch_forearm&quot;, &quot;yaw_forearm&quot;,

&quot;gyros_forearm_x&quot;, &quot;gyros_forearm_y&quot;, &quot;gyros_forearm_z&quot;,

&quot;accel_forearm_x&quot;, &quot;accel_forearm_y&quot;, &quot;accel_forearm_z&quot;,

&quot;magnet_forearm_x&quot;, &quot;magnet_forearm_y&quot;, &quot;magnet_forearm_z&quot; )

 

raw_sensor_variables &lt;- paste( raw_sensor_names, collapse=&quot;+&quot;)

f &lt;- paste(&quot;classe ~&quot;, raw_sensor_variables )</code></pre>

<p>Now multiple classification models can be tried:</p>

<pre class="r"><code>mod_rf &lt;- train( as.formula(f), data = training, method = &quot;rf&quot; )

mod_gbm &lt;- train( as.formula(f), data = training, method = &quot;gbm&quot; )

mod_lda &lt;- train( as.formula(f), data = training, method = &quot;lda&quot; )

 

test_pred_rf &lt;- predict( mod_rf, testing )

test_pred_gbm &lt;- predict( mod_gbm, testing )

test_pred_lda &lt;- predict( mod_lda, testing )

 

test_predDF &lt;- data.frame( pred_rf = test_pred_rf,

pred_gbm = test_pred_gbm,

pred_lda = test_pred_lda, class = testing$classe )

test_combModFit &lt;- train( class ~ ., method = &quot;rf&quot;, data = test_predDF )

test_combPred &lt;- predict( test_combModFit, test_predDF )</code></pre>

</div>
